📋 如何取得 GitHub Token？

  1. 前往 GitHub Settings > Tokens
  2. 點擊 "Generate new token (classic)"
  3. 選擇 "repo" 權限（完整倉庫存取權限）
  4. 設定 Token 名稱（如：TV.TCAWG.COM Admin Sync）
  5. 點擊 "Generate token" 並複製 Token
  6. 將 Token 貼到上方輸入框並儲存
