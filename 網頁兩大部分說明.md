# 本網頁兩大部分

本頁（index.html）依內容來源與維護方式，分成兩大部分：

---

## 第一部：本站影片上架（三塊）

由本站／Contentful 管理，上架與修改都在本站後台或 Contentful 完成。

| 區塊 | 說明 | 資料來源／備註 |
|------|------|----------------|
| **1. Hero 影片** | 首頁上方大畫面輪播／主視覺 | Contentful（isHero）、media.js |
| **2. 現正播出及相對時間節目** | 今日節目表、現正播出、即將播出 | Contentful、script.js；可由 admin-calendar-unified.html 上架 |
| **3. 精選節目推薦** | 精選節目卡片區 | Contentful（isFeatured）、featured.js |

---

### 第一部：如何上架？

三塊的資料都來自 **Contentful**（內容類型 `video` 或排程資料），上架方式如下。

#### 1. Hero 影片（首頁大圖輪播）

- **在 Contentful 後台**  
  1. 登入 [Contentful](https://app.contentful.com) → 進入本專案 Space。  
  2. **Content** → **Add entry** → 選 **Video**（或既有影片直接編輯）。  
  3. 填寫 **標題**、**YouTube ID**（youTubeId），並將 **isHero** 勾選為 **true**。  
  4. **Publish** 發布。  
- **用本站管理頁**：開啟 **admin-hero-videos-unified.html**，可檢查／設定哪些影片為 HERO。

#### 2. 現正播出及相對時間節目（今日節目表）

- **用本站統一節目管理**（建議）  
  1. 先啟動本地伺服器（例如雙擊 **start-server-node.bat**）。  
  2. 瀏覽器開啟 **http://localhost:8000/admin-calendar-unified.html**。  
  3. 點 **「🚀 Contentful 主控台」**，確認連線正常。  
  4. 在日曆選日期 → **「+ 添加節目」** → 填寫節目名稱、**播出時間**、**影片連結**（YouTube ID 或 MP4 網址）等。  
  5. 選好要上架的節目後，點 **「📺 上架選中節目」**，寫入 Contentful。  
- 首頁節目表會依時間從 Contentful 讀取並顯示「現正播出」「即將播出」。

#### 3. 精選節目推薦

- **在 Contentful 後台**  
  1. **Content** → **Add entry** → 選 **Video**。  
  2. 填寫 **標題**、**YouTube ID**（youTubeId），將 **isFeatured** 勾選為 **true**。  
  3. **Publish** 發布。  
- **用本站管理頁**：開啟 **admin-featured-videos.html**，可新增／編輯／設為精選；或 **admin-calendar-unified.html** 內「🎬 查看所有影片」可對既有影片設為精選。

**共通**：影片需先在 Contentful 有 **Video** 筆數（標題、YouTube ID）；Hero 看 **isHero**，精選看 **isFeatured**，節目表由排程上架。

更詳細說明見 **如何上架新影片.md**。

## 第二部：下方嵌入（影片專區）

以 **iframe** 嵌入的另一個頁面，與「本站三塊」分開。

| 項目 | 說明 |
|------|------|
| **內容** | 影片專區（所有影片／南北極／歐洲／非洲… 等分類與列表） |
| **目前來源** | 站內檔案 `video-tcawg-with-fix.html`（旅遊官方影片頁的複本） |
| **維護方式** | 若 iframe 改為指向**旅遊官方前台網址**，則由旅遊官方後台新增/修改即會反映；若維持現狀使用站內複本，則需手動更新該檔案 |
