# 發佈到網路試試

專案已連到 GitHub：`https://github.com/sammyzomb/tv.tcawg.com.git`

## 方式一：GitHub Pages（免費用、同一個 repo）

1. **先推送到 GitHub**
   - 在專案資料夾 `teavel-video-site` 開啟終端機，執行：
   ```bash
   git status
   git add .
   git commit -m "發佈：導覽列與 hero 調整、下拉半透明"
   git push origin main
   ```

2. **在 GitHub 開啟 Pages**
   - 打開：https://github.com/sammyzomb/tv.tcawg.com
   - 上方選單 **Settings** → 左側 **Pages**
   - **Source** 選 **Deploy from a branch**
   - **Branch** 選 `main`，資料夾選 **/ (root)**，按 **Save**

3. **等幾分鐘**
   - 完成後網址會是：**https://sammyzomb.github.io/tv.tcawg.com/**  
   - 若 repo 名是 `tv.tcawg.com`，也可能是：**https://sammyzomb.github.io/tv.tcawg.com/**

---

## 方式二：Netlify（拖曳資料夾即可）

1. 打開 **https://app.netlify.com**（可免費註冊）
2. 把整個 **teavel-video-site** 資料夾拖進 Netlify 的「Deploy」區
3. 部署完成會給一個隨機網址（例如 `xxx.netlify.app`），可再綁自己的網域

---

## 注意

- **header.html** 是 iframe 載入的，需與 index 同網域，用上述任一種方式發佈整包即可，不需額外設定。
- 若有用 **Contentful / API 金鑰**，確認 `browser-config.js` 不要提交敏感資訊；正式環境可改用 Netlify/ Vercel 的環境變數。
- 發佈後若導覽列或影片有異常，用瀏覽器開發者工具 (F12) 看 Console 是否有跨域或載入錯誤。
